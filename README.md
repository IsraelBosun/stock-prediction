# Stock Prediction Project

Welcome to the **Stock Prediction Project**! This repository provides tools for downloading stock price data, engineering technical indicators, fetching financial news, analyzing sentiment, and preparing datasets for machine learning-based stock price prediction.

## Features

- **Data Download**: Download historical price and volume data for top U.S. stocks (AAPL, MSFT, AMZN, GOOGL, NVDA, TSLA) using [yfinance](https://github.com/ranaroussi/yfinance).
- **Technical Indicators**: Calculate RSI, MACD, ATR, EMA, and Bollinger Bands with [ta](https://github.com/bukosabino/ta).
- **News & Sentiment**: Fetch latest financial news headlines for each stock ticker via [GNews](https://github.com/ranahaani/GNews) and analyze sentiment using [VADER](https://github.com/cjhutto/vaderSentiment).
- **Data Export**: Save processed data to CSV for further analysis or modeling.
- **Notebook Workflow**: Interactive Google Colab/ Jupyter Notebook workflow with progress bars, caching of sentiment analysis, and ready-to-use code cells.

## Notebooks

The main workflow is in [`stock_prediction.ipynb`](stock_prediction.ipynb) - see [the notebook on GitHub](https://github.com/IsraelBosun/stock-prediction/blob/main/stock_prediction.ipynb) or [run it in Google Colab](https://colab.research.google.com/github/IsraelBosun/stock-prediction/blob/main/stock_prediction.ipynb).

## Usage

1. **Clone this repository**

    ```bash
    git clone https://github.com/IsraelBosun/stock-prediction.git
    cd stock-prediction
    ```

2. **Set up your Python environment**

    - Recommended: [Google Colab](https://colab.research.google.com/)
    - Or, create a virtual environment locally.

3. **Install dependencies**

    In the notebook, install with:

    ```python
    pip install yfinance gnews torch tqdm vaderSentiment ta -q
    ```

    Or, install manually:

    ```bash
    pip install yfinance gnews torch tqdm vaderSentiment ta
    ```

4. **Run the notebook**

    Open [`stock_prediction.ipynb`](stock_prediction.ipynb) in Colab or Jupyter and follow the workflow.

## Workflow Overview

1. **Mount Google Drive** (optional, for saving cache/files).
2. **Install Python packages.**
3. **Download stock data** for selected tickers, date range 2020-01-01 to present.
4. **Save to CSV.**
5. **Calculate technical indicators** (RSI, MACD, ATR, EMA, Bollinger Width).
6. **Initialize news and sentiment analysis components.**
7. **Fetch latest financial news** for each ticker and date.
8. **Analyze sentiment** for each news headline.
9. **Attach sentiment scores** to stock data for predictive modeling.
